# 🎬 Sentiment Analysis using NLP

This project builds a **Machine Learning based Sentiment Analysis system** that classifies movie reviews as **Positive or Negative** using Natural Language Processing techniques.

---

# 🚀 Project Overview

Sentiment Analysis is widely used in:

- Product review analysis
- Social media monitoring
- Customer feedback systems
- Opinion mining

In this project we train a machine learning model using the **IMDb Movie Review Dataset** containing **50,000 movie reviews**.

---

# 📊 Dataset

Dataset used:

IMDb Movie Review Dataset

Total samples:
50,000 reviews

Classes:
Positive  
Negative
